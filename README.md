# tomnewtest

Data analysis scripts and datasets for election, economics, and occupation trend exploration.

## Contents

- Python scripts for analysis and plotting
- CSV data files
- Supporting data directories

## Quick start

1. Create and activate a virtual environment.
2. Install required Python packages.
3. Run the analysis script you need (for example, `python econ_2021_2025.py`).
